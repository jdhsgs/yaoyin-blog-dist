    Ron 12/5/2021 Utils

    # 前言

    微软的 VS Code 是一个免费的开源代码编辑器，最近越来越受欢迎。它非常轻巧、灵活，同时也提供了很多强大的功能，但也许最酷的是 VS Code 提供了规模超大的扩展插件，下面将给你介绍一些适合前端的顶级 VS Code 扩展插件。

    # 一、主题图标

    # 1.1 Material Theme

    VS Code最悠久的主题, 想要自定义编辑器主题的小伙伴可以安排起来啦！

    Material Theme插件地址 (opens new window)

    # 1.2 vscode-icons

    插件效果：

    vscode-icons插件地址 (opens new window)

    # 二、浏览器相关

    # 2.1 open in brower

    通过该插件用户可以在编辑器中快速在浏览器中打开 HTML 文件，该浏览器可以是计算机上安装的任何浏览器。

    插件效果：

    Window: 使用快捷键 Alt+B 查看效果。 Mac: 使用快捷键 option+B 查看效果。

    open in brower插件地址 (opens new window)

    # 2.2 Browser Preview

    可在编辑器中打开一个浏览器进行实时的预览，甚至调试。

    插件效果：

    Browser Preview插件地址 (opens new window)

    # 三、汉化插件

    # Chinese (Simplified) Language Pack for Visual Studio Code

    安装此插件并重新加载vscode后，你会惊奇的发现编辑器的语言从默认的英文变成中文啦，英文不好的同学不用再挠头啦！

    汉化插件地址 (opens new window)

    # 四、开发效率提升

    # 4.1 Html Language Features

    当你在书写html时，Html Language Features会提示完整的标签名称。

    插件效果：

    Html Language Features插件地址 (opens new window)

    # 4.2 Auto Close Tag

    自动闭合 HTML 标签。

    插件效果：

    Auto Close Tag插件地址 (opens new window)

    # 4.3 Auto Rename Tag

    自动重命名配对的 HTML 标签。

    插件效果：

    Auto Rename Tag插件地址 (opens new window)

    # 4.4 CSS Peek

    根据类名快速定位到CSS代码。

    插件效果：

    CSS Peek插件地址 (opens new window)

    # 4.5 IntelliSense for CSS class names in HTML

    帮助开发者提供 CSS 类名补全。

    插件效果：

    IntelliSense for CSS class names in HTML插件地址 (opens new window)

    # 4.6 lit-html

    在JavaScript/TypeScript的文件中，如果有使用到HTML标记，lit-html提供语法高亮和相应的补全支持。

    插件效果：

    lit-html插件地址 (opens new window)

    # 4.7 Todo Tree

    最实用的vscode插件，在我们的代码书写过程中经常会有需要待办的代码，比如这段代码后续需要优化，在这时可以利用todo tree做一个标记，做过的标记都会出现在插件的树视图中。等到我们达到可做后续开发的条件是，便可直接通过树视图快速定位到之前标记的代码处继续开发。

    插件效果：

    Todo Tree插件地址 (opens new window)

    # 4.8 Bookmarks

    插件效果：

    Bookmarks插件地址 (opens new window)
